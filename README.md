# TalentFlow

> Platforma rekrutacyjna nowej generacji — łącząca talenty z pracodawcami przez praktyczne wyzwania.

## O projekcie

TalentFlow to aplikacja webowa umożliwiająca firmom ocenianie kandydatów nie przez CV, ale przez realne zadania — **BattleTasks**. Pracodawcy publikują wyzwania techniczne, talenty je rozwiązują, a system wspiera cały proces decyzyjny od zgłoszenia po zatrudnienie.

## Funkcje

- Rejestracja i logowanie z podziałem na role (Talent / Pracodawca / Firma)
- System **BattleTask** — tworzenie i rozwiązywanie zadań rekrutacyjnych
- Profile talentów z odznakami i historią zgłoszeń
- Panel pracodawcy z zarządzaniem zadaniami i decyzjami
- Skrzynka wiadomości (Inbox)
- Dashboard z wizualizacją danych
- Wyszukiwanie i odkrywanie talentów
